# v0.3.20 Miss Review

Generate miss details with:

```bash
agentpack benchmark --public-suite --reproduce v0.3.20 --misses
```

Each miss should include:

- repo
- task
- missed file
- miss status
- rank
- score
- top reasons
- whether the file was ignored, low-scored, ranked but budget-cut, or absent from scan

This file is intentionally a review template until the expanded public suite is rerun and published.
